### Вопрос 1: Что из 1485 товаров — реальное?

Варианты:

- **544 реальных** — `real-products.json` это уже отфильтрованный набор
- **Ничего реального** — всё сгенерировано для демо, нужно собирать каталог с нуля
- **Частично** — часть товаров реальная (какая?)

### Вопрос 2: Откуда брать актуальный каталог?

Варианты:
- **1688.com / Taobao** — парсинг или API (OTAPI?) → нужен скрипт
- **Ручной ввод** — через админку GET2B
- **Импорт из Excel/CSV** — от реальных поставщиков
- **А-Спецторг** — забрать 604 товара оттуда (они хотя бы с нормальными поставщиками?)

### Вопрос 3: Что делать с текущими данными?

- Оставить как есть (рабочий каталог для демо)
- Почистить (убрать дубли, заполнить поставщиков)
