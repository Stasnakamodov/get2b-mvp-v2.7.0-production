# Задача: Создать красивый Debug Widget для корзины

## Контекст
У нас есть B2B каталог с корзиной. Нужен отладочный виджет, который показывает состояние корзины во время разработки.

## Текущая проблема
Сделали виджет, но он выглядит как говно - без души, без понимания дизайна, просто накидали элементов.

## Требования к виджету

### Функциональность:
- Показывает количество товаров в корзине
- Показывает активного поставщика (если есть)
- Показывает общую сумму
- Кнопка очистки корзины для тестирования
- Сворачивается/разворачивается

### Дизайн:
- **МИНИМАЛИЗМ** - никакой лишней информации
- **НЕ НУЖНО**: "Debug Mode", "DEV", "Ожидание товаров" и прочей хуйни
- Свернутый режим - плавающий шарик/кнопка
- Развернутый - компактная карточка
- Поддержка светлой и темной темы
- Градиенты фиолетовые/индиго (НЕ красные)

### Важно:
- Кнопка сворачивания, а НЕ закрытия (стрелка вниз, не крестик)
- Плавные анимации
- Продуманная композиция
- Должно выглядеть как часть премиум интерфейса

## Файлы:
- `/app/dashboard/catalog/page.tsx` - здесь нужно добавить виджет
- Состояние `debugExpanded` для управления сворачиванием
- Используем хук `useCart()` который дает `cart`, `activeSupplier`, `getTotalAmount()`, `clearCart()`

## Стек:
- Next.js 15
- Tailwind CSS
- Lucide React для иконок
- Поддержка dark mode через `dark:` классы

## Примеры хорошего дизайна:
Посмотри на Raycast, Linear, Vercel Dashboard - вот как должны выглядеть минималистичные виджеты.

## ЧТО НЕ НУЖНО:
- Не нужно делать огромные карточки
- Не нужно 100 строк информации
- Не нужно яркие цвета и кричащие элементы
- Не нужно "Debug Mode" заголовки

Сделай красиво, с душой, как будто ты дизайнер интерфейсов, а не робот который клепает компоненты.
